منصة "نادي التقنيين في سوريا"
وصف المشروع: منصة رقمية متخصصة تهدف لربط الكفاءات التقنية السورية (مبرمجين، مصممين، ومدراء مشاريع) بالشركات والجهات الباحثة عن خبرات مهنية. تعمل المنصة بآلية تشبه LinkedIn لكنها مخصصة للسوق السوري .


أبرز الميزات التي نفذتها:

إدارة الأدوار (Roles): فصل صلاحيات الحسابات بين "شركات" و"أفراد".

بيئة تفاعلية: نظام منشورات تقنية، مراسلة خاصة، وتفاعل لحظي (إعجاب وتعليقات).


محرك بحث: فلترة متقدمة للبحث عن الكفاءات بناءً على التخصص، الموقع، أو المهارات.
